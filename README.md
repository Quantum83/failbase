# 📉 Failbase

> The professional network for people who are done pretending.

Failbase is a parody LinkedIn where you post your failures instead of your wins. The worse you do, the better you look. Built in a week as part of a weekly Claude build challenge.

**[Live Demo →](https://failbase.vercel.app)** (update this link once deployed)

---

## What it does

- 📰 Public feed of failure posts with LinkedIn-style formatting
- 😬 Reactions — Yikes, Same tbh, F, Understandable
- 🏆 Shame Board — leaderboard ranked by total reactions
- ✍️ Submit your own failure
- 👤 User profiles with failure stats and badges

## Built with

- [Next.js 14](https://nextjs.org) — React framework
- [Supabase](https://supabase.com) — database
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Vercel](https://vercel.com) — deployment
